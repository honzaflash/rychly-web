Pattern Clock

This is a generative clock made in p5.js
red arrows show seconds, blue minutes and yellow hours
the partial clocks are randomly positioned in the grid with the chance inversly proportional to the shown time interval and a chance for some blank faces

Controlls:

ssave current frame
rgenerate new pattern
cswitch between modes
1set mode 1 - showing time
2set mode 2 - pointing in random directinos
